ColDog Locker

About ColDog Locker

ColDog Locker started as a security project to prerequisite the TestOut - Security Pro Certification. It developed into something bigger and helped launch ColDog Studios publicly. ColDog Locker did branch off into multiple different versions - Windows, Console, Linux, and Terminal. However, we decided to focus on prefecting a single version to be released, and removed the special version name in the process. ColDog Locker is built using the authentic Windows GUI that you are familiar with.


Basic Features

  • Custom folder name and password
  • Encrypted data
  • Failed attempts lockout system
  • View System Information

ColDog Locker for Windows



ColDog Locker is still in the process of being unified into one version. For now, you can use ColDog Locker Windows to secure your files. Be aware that ColDog Locker Windows will not be supported after ColDog Locker release.
